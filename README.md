# 🏥 LungCare AI Workstation (LungAnalyzer)
**ระบบวิเคราะห์ภาพมะเร็งปอดอัจฉริยะ สำหรับนักรังสีแพทย์**

คู่มือนี้ออกแบบมาสำหรับผู้ใช้งานทั่วไปที่ **"ไม่มีความรู้ด้านคอมพิวเตอร์"** คุณสามารถติดตั้งและใช้งานได้ใน 3 ขั้นตอนง่ายๆ ครับ

---

## 📥 ขั้นตอนที่ 1: ดาวน์โหลดโปรแกรมลงเครื่อง
1.  มองหาปุ่มสีเขียวที่เขียนว่า **"Code"** ด้านบนขวามือ
2.  คลิกแล้วเลือก **"Download ZIP"**
3.  เมื่อโหลดเสร็จ ให้ทำการ **แตกไฟล์ (Extract ZIP)** ไปไว้ที่หน้าจอ Desktop ของคุณ

---

## 🛠 ขั้นตอนที่ 2: ติดตั้งโปรแกรมพื้นฐาน (ทำครั้งเดียว)
เครื่องคอมพิวเตอร์ทั่วไปจำเป็นต้องมี "ล่าม" 2 ตัว เพื่ออ่านโค้ดโปรแกรมนี้ครับ ให้ดาวน์โหลดและติดตั้งตามลิงก์ด้านล่างนี้:

### 1. ล่ามสำหรับหน้าเว็บ (Node.js)
*   **ดาวน์โหลด:** [กดที่นี่เพื่อโหลด Node.js (เลือกตัวทางซ้ายที่เขียนว่า LTS)](https://nodejs.org/en/download/)
*   **วิธีติดตั้ง:** กด Next ไปเรื่อยๆ จนจบ (ไม่ต้องตั้งค่าอะไรพิเศษ)

### 2. ล่ามสำหรับ AI (Python)
*   **ดาวน์โหลด:** [กดที่นี่เพื่อโหลด Python](https://www.python.org/downloads/)
*   **วิธีติดตั้ง:** 
    > [!IMPORTANT]
    > **สำคัญมาก:** ในหน้าแรกของการติดตั้ง **ต้องติ๊กถูกที่ช่อง "Add Python to PATH"** (อยู่ด้านล่างสุด) ก่อนกด Install Now มิฉะนั้นโปรแกรมจะทำงานไม่ได้ครับ

### 3. เตรียมกุญแจ AI (Gemini API Key)
เพื่อให้ระบบ Cloud AI ทำงานได้ คุณจำเป็นต้องมีกุญแจรหัส (API Key):
*   **วิธีรับกุญแจ:** เข้าไปที่ [Google AI Studio](https://aistudio.google.com/app/apikey) แล้วกดสร้างกุญแจฟรีครับ
*   **วิธีใส่รหัส:** ไม่ต้องกังวลครับ เมื่อคุณเปิดโปรแกรมครั้งแรก (ในขั้นตอนที่ 3) **ระบบจะถามรหัสนี้จากคุณโดยอัตโนมัติ** เพียงแค่เตรียมก๊อปปี้รหัสรอไว้ก็พอครับ

---

## 🚀 ขั้นตอนที่ 3: เริ่มใช้งาน (เปิดโปรแกรม)
หลังจากติดตั้งขั้นตอนที่ 2 เสร็จแล้ว คุณไม่จำเป็นต้องเขียนโค้ดเองครับ ผมทำ "ปุ่มเปิด" ไว้ให้แล้ว:

### สำหรับผู้ใช้ Windows (PC ทั่วไป)
1.  เข้าไปในโฟลเดอร์โปรแกรมที่คุณแตกไฟล์ไว้
2.  หาไฟล์ที่ชื่อ **`start_windows.bat`**
3.  **ดับเบิลคลิก** ที่ไฟล์นี้ครั้งเดียว ระบบจะทำการเตรียมเครื่องและเปิดหน้าเว็บให้คุณโดยอัตโนมัติครับ

### สำหรับผู้ใช้ macOS (เครื่อง Mac)
1.  เข้าไปในโฟลเดอร์โปรแกรม
2.  หาไฟล์ที่ชื่อ **`start_mac.command`**
3.  **ดับเบิลคลิก** เพื่อเริ่มต้นใช้งาน

> [!TIP]
> **หากเครื่อง Mac แจ้งว่าเปิดไม่ได้เพราะไม่มีสิทธิ์ (Permission Denied):**
> 1. เปิดโปรแกรม **Terminal** ในเครื่องคุณ
> 2. พิมพ์คำสั่ง `chmod +x ` (เว้นวรรค 1 ที) แล้วลากไฟล์ `start_mac.command` มาวางใน Terminal
> 3. กด Enter แล้วลองดับเบิลคลิกเปิดไฟล์ใหม่อีกครั้งครับ

---

## 🧠 หลักการทำงาน (แบบเข้าใจง่าย)
โปรแกรมนี้ทำงานเหมือน "ห้องแล็บแพทย์ส่วนตัว" ในเครื่องคุณ:
1.  **หน้าจอ:** คือที่ที่คุณใช้อัปโหลดรูปและวาดเส้นก้อนเนื้อ
2.  **สมองกล AI:** เมื่อคุณกดปุ่มวิเคราะห์ ข้อมูลจะถูกส่งไปประมวลผลผ่าน "สมองกล" (AI Model) ที่เราเตรียมไว้ให้ ซึ่งจะให้คำตอบออกมาเป็นความเสี่ยงเปอร์เซ็นต์และคำแนะนำเบื้องต้นครับ

> [!TIP]
> **การใช้งาน:** ขณะใช้งานโปรแกรม **ห้ามปิดหน้าต่างดำๆ (Terminal/Command Prompt)** ที่เด้งขึ้นมานะครับ เพราะนั่นคือหัวใจที่ทำให้โปรแกรมทำงานอยู่ครับ หากใช้งานเสร็จแล้วค่อยปิดหน้าต่างนั้นได้เลย

> **หมายเหตุ:** โปรแกรมนี้เป็นเพียงเครื่องมือช่วยสนับสนุนการตัดสินใจเบื้องต้น ผลการวินิจฉัยที่ถูกต้องแม่นยำที่สุดต้องมาจากความเห็นของรังสีแพทย์ผู้เชี่ยวชาญเท่านั้นครับ
