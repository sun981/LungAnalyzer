# 🏥 LungCare AI Workstation (LungAnalyzer)

**LungCare AI Workstation** คือระบบช่วยเหลือนักรังสีแพทย์ในการวิเคราะห์ก้อนเนื้อในปอด (Lung Nodule Classification) จากภาพ CT Scan โดยใช้ขุมพลัง AI แบบคู่ขนาน (Dual-Engine) 

## 🌟 จุดเด่นของระบบ
*   **Dual-Engine AI:** เลือกประมวลผลได้ทั้งผ่าน Cloud (Gemini 3.1 Flash) เพื่อความฉลาดเชิงลึก หรือประมวลผลผ่าน Local AI ในพื้นที่ (HuggingFace) เพื่อความเป็นส่วนตัวสูงสุด (Privacy)
*   **Radiomics Analysis:** ระบบคำนวณค่าทางคณิตศาสตร์ของก้อนเนื้อให้อัตโนมัติ (Area, Circularity, Density, Heterogeneity)
*   **Clinical Interface:** หน้าออกแบบตามมาตรฐานสถานีงานแพทย์ (Medical Workstation) ใช้งานง่าย รองรับการวาด Mask และปรับแต่ง Contrast (Windowing)
*   **Edge Deployment:** รองรับการติดตั้งบนฮาร์ดแวร์ขนาดเล็กอย่าง Raspberry Pi 5 สำหรับใช้ในคลินิกหรือโรงพยาบาล

## 🧠 หลักการทำงานของระบบ (How it works?)

เพื่อให้แพทย์หรือผู้ใช้งานทั่วไปเข้าใจง่ายขึ้น ระบบ **LungCare AI** ทำงานเหมือนกับ "ห้องแล็บรังสีดิจิทัล" ที่แบ่งหน้าที่กันดังนี้ครับ:

### 1. ส่วนหน้าจอ (Frontend - Next.js)
เปรียบเสมือน **"โต๊ะทำงานของแพทย์"** ที่ใช้สำหรับเปิดดูรูปภาพ CT Scan, วาดเส้นรอบก้อนเนื้อ (Segmentation) และปรับแต่งภาพให้ชัดเจนขึ้น ข้อมูลทั้งหมดที่คุณทำจะแสดงผลที่นี่ในแบบเรียลไทม์

### 2. ส่วนการวิเคราะห์เชิงตัวเลข (Radiomics Analysis)
เมื่อคุณวาดรูปทรงของก้อนเนื้อเสร็จ ระบบจะไม่เพียงแค่ดูด้วยตาเปล่า แต่จะ **"คำนวณสถิติ"** ให้ด้วย เช่น:
*   ก้อนเนื้อกลมแค่ไหน? (ถ้าขรุขระมาก อาจเสี่ยงเป็นเนื้อร้ายสูง)
*   ก้อนเนื้อทึบแค่ไหน?
*   ความสม่ำเสมอของเนื้อใน (ถ้าเนื้อข้างในไม่สม่ำเสมอ เป็นสัญญาณที่ควรระวัง)

### 3. ส่วนสมองกล AI (Dual-Engine AI Brain)
เมื่อคุณกดวิเคราะห์ ระบบจะส่งข้อมูลภาพและตัวเลขไปให้ "สมองกล" ตัวใดตัวหนึ่งที่คุณเลือก:
*   **Cloud AI (Gemini):** เปรียบเหมือนส่งไปปรึกษา "ผู้เชี่ยวชาญบนระบบคลาวด์" ที่มีความฉลาดล้ำลึกและให้คำตอบที่เป็นเหตุเป็นผล
*   **Local AI (HuggingFace):** เปรียบเหมือนมี "ผู้เชี่ยวชาญที่นั่งอยู่ข้างโต๊ะคุณ" (ในเครื่องของคุณเอง) ทำงานได้รวดเร็วแม้ไม่มีอินเทอร์เน็ต และรักษาความลับของคนไข้ได้ดีที่สุดเพราะข้อมูลไม่หลุดออกนอกตึก

## 🏗 สถาปัตยกรรม (Architecture)
*   **Frontend:** Next.js 14, Tailwind CSS, Fabric.js (Canvas Handling)
*   **Cloud AI Proxy:** API Routes ใน Next.js เชื่อมต่อกับ Google Gemini API
*   **Local AI Service:** FastAPI (Python) รันโมเดล HuggingFace (PyTorch)
